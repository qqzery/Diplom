# Розробка адаптивного вебресурсу для ветеринарної клініки "VetDomik" з використанням сучасних технологій фронтенд-розробки
Кваліфікаційна робота на підтвердження ступеня фахового молодшого бакалавра ([ВСП «ППФК НТУ «ХПІ»](http://polytechnic.poltava.ua)). 

Керівник роботи – **Барабан Дмитро Миколайович**.
## Завдання до дипломної роботи
Продукт має являти собою односторінковий вебзастосунок з високим рівнем продуктивності та унікальним дизайном. Семантична верстка HTML5, стилізація за допомогою CSS3 (або препроцесорів Sass/Less), мова програмування JavaScript (можливе використання бібліотек/фреймворків). Сайт повинен містити інтерактивні елементи (галереї, форми, анімації), бути повністю адаптивним та оптимізованим для швидкого завантаження. За основу взяти структуру та контент сайту https://barkat.com.ua/
## Використані технології
* HTML5 — створення семантичної структури вебсторінок.
* CSS3 — оформлення інтерфейсу та адаптивна верстка.
* CSS Flexbox — побудова одновимірних макетів і вирівнювання елементів.
* CSS Grid Layout — створення адаптивних двовимірних сіток.
* JavaScript (ES6+) — реалізація інтерактивності та клієнтської логіки.
* Fetch API — надсилання даних форми до зовнішніх сервісів.
* FormData — збір та обробка даних користувацьких форм.
* Serverless Form Handlers (Web3Forms / Formspree) — обробка заявок без власного серверного бекенду.
* Regular Expressions (RegExp) — валідація введених користувачем даних.
* Visual Studio Code (VS Code) — середовище розробки.
* Live Server — локальний запуск сайту та автоматичне оновлення сторінок.
* Prettier — автоматичне форматування програмного коду.
* Google Lighthouse — тестування продуктивності, доступності та SEO.
* Git — система контролю версій.
* GitHub Pages / Netlify / Vercel — розгортання та хостинг вебресурсу.
* WebP — оптимізований формат зображень для швидкого завантаження сторінок.
